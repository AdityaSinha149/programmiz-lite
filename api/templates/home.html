Write and run .psa programs in the browser

Programmiz Lite is a lightweight workspace for .psa files. The language is C-like, but it has limits and only works with the compiler binary provided for this project.

What you can do

Create folders, organize files, open the editor, save code, and run .psa programs with interactive input.

About .psa

It looks like C, but it is not full C. Some language features, libraries, and system behavior may be limited or unsupported.

Output

Only .psa files get a run button and output panel. Other files can still be edited and saved, but they are not executable here.

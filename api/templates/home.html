Programmiz Lite

Write and run .psa programs in the browser

This is a lightweight programming workspace for .psa files. A .psa file uses a C-like syntax, but it is intentionally limited and tied to the compiler binary provided with this project.

What you can do

Create folders, organize files, open a built-in editor, save code, and run .psa programs with interactive input.

About .psa

It is C-like, but not full C. Some language features, libraries, or system-level behavior may be unavailable depending on the compiler rules.

Output

Only .psa files show compile/run output. Other files can still be edited and saved, but they do not have a run panel.

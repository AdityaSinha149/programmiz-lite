Write and run .psa programs in the browser

Programmiz Lite is a lightweight workspace for .psa files. The language is a restricted C-like subset with clear supported and unsupported features.

What works in .psa

  • Types: int, char, void
  • Functions: declarations/prototypes and function definitions
  • Scopes: global declarations and block scopes
  • Statements: expression statements, if/else, while, return, nested blocks
  • Expressions: assignment, arithmetic, relational, equality, logical AND/OR, unary - and !
  • Calls: function calls with argument lists
  • Literals: integer and char literals

What is not supported

  • No struct, union, or enum
  • No pointers
  • No typedef
  • No switch
  • Not full C standard library compatibility

Editor and runtime behavior

  • Only files ending with .psa show the Run button and output panel
  • Other file types can be edited and saved, but not compiled/executed
  • Program input is passed through the input box under the editor
